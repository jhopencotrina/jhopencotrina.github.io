Semana 2: HTML

DEFINICIONES

HTML es un lenguaje de marcado que se utiliza para estructurar el contenido de una página web. Consiste en una serie de elementos o etiquetas que rodean el contenido y le dan significado.

Las etiquetas HTML son códigos encerrados entre corchetes angulares <>, y generalmente vienen en pares de apertura <> y cierre.

Para que sirve HTML:

1. Estructurar la Información: HTML se utiliza para organizar el contenido de una página web en diferentes secciones, como encabezados, párrafos, listas y tablas, lo que facilita la comprensión y presentación de la información.

2. Creación de Hipervínculos: HTML permite la creación de enlaces o hipervínculos que conectan páginas web entre sí, permitiendo a los usuarios navegar de una página a otra.

3. Compatibilidad con Navegadores: HTML es un estándar ampliamente aceptado y compatible con la mayoría de los navegadores web, lo que garantiza que las páginas web se visualicen correctamente en diversas plataformas y dispositivos.

PROCEDIMIENTOS

Utilizar HTML implica seguir una serie de procedimientos para crear y estructurar el contenido de una página web.

Los principales pasos son:

Crear un Documento HTML: Se abre en editor de texto preferido (como Notepad, Visual Studio Code, Sublime Text).

Luego se le agrega el contenido entre las etiquetas como body y /body, y se agrega el contenido de la página utilizando diversas etiquetas HTML.

Se puede añadir enlaces e incorporar imagenes, se utiliza la etiqueta a para crear enlaces a otras páginas web o recursos. El atributo href contiene la URL del enlace. Y para las imagenes se usa la etiqueta "img" para, el atributo src contiene la ruta de la imagen y el atributo alt proporciona un texto alternativo.

Se crea Listas usando las etiquetas ul y li para listas no ordenadas, y ol y li para listas ordenadas.

Por ultimo, se puede aplicar Estilos con CSS. Si deseas estilizar tu página, puedes enlazar un archivo de estilo CSS utilizando la etiqueta link en la sección head.

RESULTADOS

Se aprendio que las páginas web desarrolladas con HTML tienen una estructura coherente y organizada, además de facilitar la comprensión y navegación para los usuarios y motores de búsqueda.

Las páginas HTML son compatibles con la gran mayoría de los navegadores web, lo que asegura que el contenido se vea y funcione de manera consistente en diferentes plataformas y dispositivos.

Aprender a planificar y gestionar proyectos web desde la concepción hasta el despliegue.

Por ultimo, se aprendio que las páginas bien estructuradas en HTML pueden ser más fácilmente indexadas por los motores de búsqueda para mejorar la visibilidad en los resultados de búsqueda y la accesibilidad para los usuarios.

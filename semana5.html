Semana 5: Desarrollo con React

DEFINICIONES

React es una libreria de JavaScript que sirve para crear interfaces de usuario, tambien se utiliza para crear SPA aplicaciones de una sola pagina

React se utiliza para una variedad de propósitos en el desarrollo web y móvil como:

Desarrollo de aplicaciones web interactivas ya que permite crear interfaces de usuario dinámicas y altamente interactivas para aplicaciones web

Facilita la manipulación y modificación de elementos HTML y CSS en tiempo real.

Tambien el desarrollo de aplicaciones móviles utilizando React Native y una integración con otras bibliotecas y frameworks

PROCEDIMIENTOS

Trabajar con React implica una serie de procedimientos que abarcan desde la configuración inicial del entorno de desarrollo hasta la creación de componentes y el manejo del estado.

Para trabajar con React se realiza los siguientes paso:

Se instala Node.js y se ejecuta el comando: npx create-react-app mi-aplicacion-react

Se puede explorar la estructura de carpetas y archivos generada por Create React App.

Se puede crear componentes de React utilizando diferentes funciones que en las siguientes semanas se explicara.

RESULTADOS

Se observó que el uso de React en el desarrollo de páginas web puede llevar a varios resultados positivos que benefician tanto a los desarrolladores como a los usuarios.

React facilita el mantenimiento del código y acelera el desarrollo al poder utilizar componentes en diferentes partes de la aplicación.

Tambien facilita el manejo de la lógica de la aplicación y la actualización dinámica de la interfaz de usuario en respuesta a cambios de estado.

Por último, React cuenta con un amplio ecosistema de bibliotecas y herramientas, como Redux para la gestión del estado, y muchas más.

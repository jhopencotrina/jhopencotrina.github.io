  • Introducción a React

    React es una biblioteca de JavaScript para crear interfaces de usuario, React se usa para crear aplicaciones de una sola página, tambien permite crear componentes de la interfaz de usuario reutilizables.


    ¿Cómo funciona React?

       React crea un DOM VIRTUAL en la memoria.
       En lugar de cambiar de manipular el DOM del navegador directamente, React crea un DOM virtual en la memoria donde realiza la manipulación necesaria antes de manipular el DOM del navegador.
       React solo cambia lo que necesita ser cambiado.
       React descubre que los cambios se han realizado y cambia solo lo que debe cambiarse.

CONCLUSIONES

La creación de un portafolio web es una buena practica para cualquier a desarrollador web. Al utilizar tecnologías como HTML, CSS, JavaScript y React, se pueden lograr resultados impresionantes y funcionales que te permiten desarrollar tu experiencia y creatividad en el campo.

1. Mostrar habilidades técnicas: La construcción de un portafolio web utilizando HTML y CSS dsirve como ejercicio para el desarrollo web. La incorporación de JavaScript y React muestra una comprensión avanzada de la interactividad y la creación de aplicaciones web modernas.

2. Demostrar creatividad: El diseño de un portafolio es una oportunidad para destacar tus habilidades de diseño web. Puedes mostrar tu capacidad para crear interfaces atractivas y amigables para el usuario.

3. Mejorar tu presencia en línea: Un portafolio web bien diseñado puede aumentar tu visibilidad en línea y ayudarte a destacar en un mercado competitivo.

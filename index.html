TABLA DE CONTENIDO

PRIMER CONSOLIDADO

Semana 1: Tecnologias de Desarrollo Web

Las tecnologías de desarrollo web se refieren a un conjunto de herramientas, lenguajes de programación, frameworks y estándares utilizados para crear aplicaciones y sitios web en Internet.

CONTENIDO DE LA SEMANA 1

Semana 2: HTML

HTML es un lenguaje estandar que se utiliza para definir la estructura y el formato de una página web, lo que permite la presentación de texto, imágenes, enlaces, formularios y otros elementos multimedia en un navegador web.

CONTENIDO DE LA SEMANA 2

Semana 3: CSS y Liberias

CSS es un lenguaje utilizado para controlar la presentación y el diseño de páginas web, que se se centra en aspectos visuales como colores, fuentes, márgenes, tamaños y posiciones de elementos en la página.

CONTENIDO DE LA SEMANA 3

Semana 4: Java Script

JavaScript es un lenguaje de programación de alto nivel, interpretado por los navegadores web, que se utiliza para hacer que las páginas web sean interactivas y dinámicas.

CONTENIDO DE LA SEMANA 4

Semana 5: Desarrollo con React

React es una libreria de JavaScript que sirve para crear interfaces de usuario, tambien se utiliza para crear SPA aplicaciones de una sola pagina

CONTENIDO DE LA SEMANA 5

Semana 6: Desarrollo de aplicaciones con Eventos, Condicionales y Bucles con Framework JS

Desarrollar aplicaciones con eventos, condicionales y bucles con un framework JavaScript implica utilizar las características proporcionadas por un framework específico para manejar la interacción del usuario, realizar acciones condicionales basadas en lógica de programación y ejecutar repeticiones eficientes.

CONTENIDO DE LA SEMANA 6

Semana 7: Uso de Hooks

Los Hooks en React son funciones que permiten a los componentes funcionales tener características de los componentes de clase, como el estado y los ciclos de vida, sin necesidad de convertirlos en clases.

CONTENIDO DE LA SEMANA 7

SEGUNDO CONSOLIDADO

Semana 9: Next JS CSR / SSR

Next.js es un framework de desarrollo web React que simplifica y agiliza la creación de aplicaciones web.

CONTENIDO DE LA SEMANA 9

Semana 10: Consumo de APIs dentro de Next.Js

En Next.js, el consumo de APIs se refiere a la práctica de hacer solicitudes a interfaces de programación de aplicaciones (APIs, por sus siglas en inglés) desde las páginas o componentes de tu aplicación Next.js.

CONTENIDO DE LA SEMANA 10

Semana 11: Next Auth y Server Side Props en Next.js

NextAuth.js es una biblioteca de autenticación para Next.js que facilita la implementación de sistemas de autenticación en tus aplicaciones y GetServerSideProps es una función en Next.js que se utiliza para obtener datos en el lado del servidor (SSR) antes de que una página se renderice.

CONTENIDO DE LA SEMANA 11

Semana 12: Desarrollo backend

El desarrollo backend, se refiere a la creación y mantenimiento de la parte de una aplicación web o software que opera en el servidor y se encarga de la lógica empresarial, la manipulación de datos y la interacción con la base de datos.

CONTENIDO DE LA SEMANA 12

Semana 13: Desarrollar Aplicaciones con Spring Core y Spring Boot

El desarrollo de aplicaciones con Spring Core y Spring Boot se refiere al uso de dos proyectos clave dentro del ecosistema de Spring Framework, una plataforma de desarrollo para la creación de aplicaciones empresariales en Java.

CONTENIDO DE LA SEMANA 13

Semana 14: Desarrollar API Rest con Spring

Desarrollar una API REST con Spring implica utilizar el conjunto de herramientas proporcionado por el framework Spring para construir servicios web que sigan el estilo arquitectónico REST

CONTENIDO DE LA SEMANA 14

CONCLUSIONES

La creación de un portafolio web que destaca proyectos y habilidades en el desarrollo de páginas web es una inversión valiosa para cualquier profesional o aspirante a desarrollador web. Al utilizar tecnologías como HTML, CSS, JavaScript, React, Next.js, APIs y Spring se pueden lograr resultados impresionantes y funcionales que muestren tu experiencia y creatividad en el campo.

1. Mostrar habilidades técnicas: La construcción de un portafolio web utilizando HTML y CSS demuestra un dominio sólido de las bases del desarrollo web. La incorporación de JavaScript y React muestra una comprensión avanzada de la interactividad y la creación de aplicaciones web modernas.

2. Demostrar creatividad: El diseño de tu portafolio es una oportunidad para destacar tus habilidades de diseño web. Puedes mostrar tu capacidad para crear interfaces atractivas y amigables para el usuario.

3. Mejorar tu presencia en línea: Un portafolio web bien diseñado puede aumentar tu visibilidad en línea y ayudarte a destacar en un mercado competitivo.

4. React proporciona un enfoque claro para gestionar el estado de la aplicación, lo que facilita la manipulación y actualización de datos de manera eficiente.

5. React y Next.js permiten un desarrollo eficiente y modular mediante la creación de componentes reutilizables. Esto simplifica la gestión del código y facilita la escalabilidad del proyecto.

6. React y Next.js se integran fácilmente con APIs, permitiendo un consumo eficiente de datos desde el backend.

7. Spring es conocido por sus características de seguridad robustas, lo que garantiza que las aplicaciones web desarrolladas con esta tecnología cumplan con los estándares de seguridad.

8. Utilizar tecnologías como React, Next.js, APIs y Spring permite una mayor adaptabilidad a los cambios tecnológicos, ya que son tecnologías bien respaldadas y evolucionan con el tiempo.

¿QUÉ APRENDÍ Y CÓMO?

El proceso de aprendizaje de la creación de una página web utilizando HTML, CSS, JavaScript, React, Next.js, APIs y Spring ha sido continuo y gratificante. Sumergirme en estos lenguajes y tecnologías me ha proporcionado una comprensión profunda de varios aspectos clave:

Semana 1 y 2: Estructura y marcado con HTML

Aprendí a utilizar HTML para establecer una estructura sólida en mis páginas web, organizando el contenido de manera semántica y utilizando códigos Emmet para simplificar las tablas de manera eficiente.

Semana 3: Diseño con CSS

CSS me permitió explorar la creatividad en el diseño web. Aprendí a aplicar estilos a elementos HTML y a diseñar interfaces atractivas y funcionales, entendiendo conceptos clave como la cascada, la herencia y la especificidad de CSS.

Semana 4: Interactividad con JavaScript

JavaScript abrió la puerta a la interactividad y dinamismo en mis proyectos. Aprendí a crear funciones y eventos, lo que me permitió responder a las acciones del usuario y manipular el contenido en tiempo real.

Semana 5: Desarrollo de aplicaciones con React

Utilizar React me proporcionó conocimientos para construir aplicaciones web más complejas y dinámicas. Aprendí a dividir la interfaz en componentes reutilizables, facilitando la organización y el mantenimiento de proyectos grandes. La gestión de estados y el enrutamiento en React también se convirtieron en habilidades clave.

Semana 6: Desarrollo de aplicaciones con Eventos, Condicionales y Bucles con Framework JS

Al incorporar Java dentro de React, exploré el uso de diferentes condicionales y bucles. La comprensión de estos elementos fue respaldada por la guía de diversos materiales, facilitando la comprensión de la programación.

Semana 7: Uso de Hooks

El uso de hooks como useState, useEffect y useContext me enseñó cómo separar los componentes en React y Next.js. Utilizar estos hooks con diversas bibliotecas contribuyó a mejorar la interfaz de la página web de manera efectiva.

Semana 8: Evaluación Parcial del Consolidado I

Durante la evaluación parcial, que consistió en la creación de una ruleta rusa con diferentes botones, amplié mis conocimientos en programación JavaScript utilizando Canvas, mejorando la interfaz e interacción de la página web.

Semana 9: Next JS CSR / SSR

Explorar Next.js me permitió comprender las diferencias con React, especialmente en el uso de APIs y la organización de carpetas. Aprendí sobre las diversas funcionalidades que ofrece Next.js.

Semana 10: Consumo de APIs dentro de Next.Js

Adquirí habilidades para utilizar diferentes APIs, como "Rick and Morty", "Breaking Bad", "Pokemon" y "Marvel", lo cual es beneficioso para proyectos que requieren el consumo de estas API en el desarrollo de páginas web con Next.js y React.

Semana 11: Next Auth y Server Side Props en Next.js

Obtuve conocimientos sobre las propiedades ofrecidas por Next.js, así como su creación y ejecución dentro de Visual Studio. Esto contribuye a una mejor creación en el desarrollo de páginas web.

Semana 12: Desarrollo backend

Aprendí a distinguir entre el desarrollo frontend y backend, centrándome en el consumo de archivos JSON y bases de datos en el backend.

Semana 13: Desarrollar Aplicaciones con Spring Core y Spring Boot

Exploré el uso de Spring dentro de Next.js y su conexión con Java, mejorando la comprensión de la relación entre estas tecnologías.

Semana 14: Desarrollar API Rest con Spring

En la última fase del curso, aprendí a utilizar APIs con Spring para el desarrollo de páginas web, lo cual fue fundamental para trabajar con mi grupo en el proyecto final, que involucró tanto el frontend como el backend de una página web.
